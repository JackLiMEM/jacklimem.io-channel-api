# 发送短信接口文档

# 1. 接口说明 {#接口说明}

**本接口主要针对借款前发送短信验证码。**

借款确认前调用此接口处理保险签约和连资贷签约相关短信发送业务

# 2. 接口 URL
> 请求方式: post

> 请求地址: ${domain}/api/h5/openapi/borrow/sendBorrowConfirmSms

# 3. 请求参数 {#请求参数}

| 参数 | 类型 | 是否必选 | 描述 |
| :--- | :--- | :--- | :--- |
| orderNo | string | 是 | 我方订单号 |
| optType | int | 是 | 操作类型1支付协议签约2保险签约 |


# 4. 响应参数 {#响应参数}

| 参数 | 类型 | 是否必选 | 描述 |
| :--- | :--- | :--- | :--- |
| code | integer | 是 | 响应码，如200|
| msg | integer | 是 | 响应信息，如成功 |
| data | Object| 是 | json数据|
